# 📦 数据导入导出指南

## 🎯 功能说明

通过导入导出功能，您可以：
- ✅ 备份您的数据
- ✅ 在不同设备间迁移数据
- ✅ 与他人共享课程数据
- ✅ 恢复误删除的数据

---

## 📤 导出数据

### 操作步骤

1. **打开设置页面**
   - 点击底部导航栏的"设置"标签

2. **点击导出数据**
   - 在"数据管理"区域
   - 点击"导出数据"按钮

3. **保存文件**
   - 浏览器会自动下载 JSON 文件
   - 文件名格式：`lesson-data-时间戳.json`
   - 例如：`lesson-data-1709467200000.json`

### 导出的数据包含

```json
{
  "packages": [
    {
      "id": "课程包ID",
      "courseType": "课程类型",
      "courseName": "课程名称",
      "totalHours": 48,
      "remainingHours": 47,
      "purchaseDate": "2023-11-28",
      "createTime": "创建时间",
      "updateTime": "更新时间"
    }
  ],
  "records": [
    {
      "id": "记录ID",
      "packageId": "课程包ID",
      "courseType": "课程类型",
      "courseName": "课程名称",
      "recordType": "consume",
      "consumeHours": 1,
      "recordDate": "2024-01-25",
      "remark": "备注",
      "createTime": "创建时间"
    }
  ],
  "exportTime": "2024-03-03T12:00:00.000Z"
}
```

## 📥 导入数据

### 操作步骤

1. **打开设置页面**
   - 点击底部导航栏的"设置"标签

2. **点击导入数据**
   - 在"数据管理"区域
   - 点击"导入数据"按钮

3. **选择文件**
   - 选择之前导出的 JSON 文件
   - 必须是本应用导出的格式

4. **选择导入方式**

   **方式1：合并数据（推荐）**
   ```
   点击"确定"
   
   效果：
   - 保留现有数据
   - 只导入新的课程包和记录
   - 跳过重复的数据（根据ID判断）
   ```

   **方式2：覆盖数据**
   ```
   点击"取消"
   
   效果：
   - 清空现有数据
   - 完全使用导入的数据
   - 适合全新开始或完整恢复
   ```

5. **确认导入**
   - 看到成功提示
   - 页面自动刷新
   - 新数据已显示

## 🔄 使用场景

### 场景1：备份数据

**操作：**
1. 定期导出数据
2. 保存到云盘或其他安全位置
3. 建议每周备份一次

**文件命名建议：**
```
lesson-backup-2024-03-03.json
lesson-backup-2024-03-10.json
lesson-backup-2024-03-17.json
```

### 场景2：更换设备

**从旧设备迁移到新设备：**

**在旧设备上：**
1. 打开应用设置
2. 导出数据
3. 保存文件到云盘或发送给自己

**在新设备上：**
1. 打开应用
2. 进入设置
3. 导入数据
4. 选择"覆盖数据"
5. 完成！

### 场景3：多人共享数据

**老师分享给学生：**
1. 老师导出课程数据
2. 发送给学生
3. 学生导入数据
4. 选择"合并数据"（如果学生已有自己的数据）

### 场景4：恢复误删数据

**如果不小心删除了数据：**
1. 找到之前的备份文件
2. 导入数据
3. 选择"覆盖数据"
4. 数据恢复完成

## ⚠️ 注意事项

### 导入前

- ✅ 确保文件是本应用导出的 JSON 格式
- ✅ 检查文件大小是否合理
- ✅ 如有重要数据，先导出当前数据备份
- ✅ 了解"合并"和"覆盖"的区别

### 导入时

**合并数据：**
- ✅ 安全，不会丢失现有数据
- ✅ 只添加新的内容
- ✅ 重复的数据会被跳过
- ⚠️ 可能导致数据冗余

**覆盖数据：**
- ⚠️ 会清空现有所有数据
- ✅ 适合完整恢复
- ✅ 避免数据重复
- ❌ 现有数据会丢失

### 导入后

- ✅ 检查数据是否正确
- ✅ 验证课程包数量
- ✅ 验证记录数量
- ✅ 测试功能是否正常

## 🔍 常见问题

### Q1: 导入后数据显示不对？

**解决：**
1. 刷新页面（Cmd+R 或 Ctrl+R）
2. 清除浏览器缓存
3. 重新导入数据

### Q2: 导入提示"数据格式错误"？

**原因：**
- 文件不是本应用导出的格式
- 文件已损坏
- 手动修改了文件内容

**解决：**
1. 确认文件来源
2. 重新导出原始数据
3. 不要手动编辑 JSON 文件

### Q3: 导入后课程包显示但记录消失？

**原因：**
- 记录的 `packageId` 与课程包的 `id` 不匹配

**解决：**
1. 使用"覆盖数据"方式重新导入
2. 确保导出时数据完整

### Q4: 可以导入多个文件吗？

**可以！**

使用"合并数据"方式：
1. 导入第一个文件（合并）
2. 导入第二个文件（合并）
3. 导入第三个文件（合并）
4. 所有数据会合并在一起

### Q5: 如何只导入部分数据？

**方法：**
1. 用文本编辑器打开 JSON 文件
2. 删除不需要的数据
3. 保存文件
4. 导入修改后的文件

**注意：**
- 确保 JSON 格式正确
- 不要破坏数据结构
- 建议先备份原文件

### Q6: 导入后重复了怎么办？

**如果使用了"覆盖数据"：**
- 不会有重复，数据完全替换

**如果使用了"合并数据"：**
- 相同ID的数据会被跳过
- 不同ID的数据会被添加
- 不会产生重复

### Q7: 可以在手机上导入导出吗？

**导出：**
- ✅ 可以，文件会下载到手机

**导入：**
- ✅ 可以，使用文件选择器
- 需要先将文件保存到手机
