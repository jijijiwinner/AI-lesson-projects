# ⚡ 导入导出功能 - 3分钟上手

## 🎯 快速开始

### 📤 导出数据（备份）

**3步完成：**

1. 打开"设置"页面
2. 点击"导出数据"
3. 保存下载的 JSON 文件

✅ 完成！文件名类似：`lesson-data-1709467200000.json`

---

### 📥 导入数据（恢复/迁移）

**4步完成：**

1. 打开"设置"页面
2. 点击"导入数据"
3. 选择之前导出的 JSON 文件
4. 选择导入方式：
   - **点"确定"** = 合并数据（保留现有 + 添加新的）
   - **点"取消"** = 覆盖数据（清空现有 + 完全替换）

✅ 完成！页面自动刷新显示新数据

---

## 🔄 常用场景

### 场景1：定期备份

```
每周一次：
设置 → 导出数据 → 保存到云盘
```

### 场景2：换手机/电脑

```
旧设备：
设置 → 导出数据 → 发送给自己

新设备：
设置 → 导入数据 → 选"覆盖数据"
```

### 场景3：分享数据给他人

```
发送者：
设置 → 导出数据 → 发送文件

接收者：
设置 → 导入数据 → 选"合并数据"
```

---

## ⚠️ 重要提示

### 合并 vs 覆盖

**合并数据（点"确定"）：**
- ✅ 安全，不会丢失现有数据
- ✅ 只添加新内容
- ✅ 跳过重复数据

**覆盖数据（点"取消"）：**
- ⚠️ 清空所有现有数据
- ✅ 完全使用导入的数据
- 适合：换设备、全新开始
