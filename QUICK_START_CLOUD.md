# ⚡ 云同步快速配置（3分钟完成）

## ✅ 代码已切换完成

所有页面已自动切换到云同步版本，现在只需配置 uni-cloud 即可使用！

## 🚀 方式一：使用 HBuilderX（最简单，推荐）

### 1. 下载 HBuilderX
访问：https://www.dcloud.io/hbuilderx.html
下载并安装（免费）

### 2. 创建云服务空间（2分钟）
1. 用 HBuilderX 打开本项目
2. 右键点击项目根目录的 `uniCloud-aliyun` 文件夹
3. 选择 "创建云服务空间"
4. 填写：
   - 空间名称：`lesson-tracking`（或其他名称）
   - 服务商：选择 "阿里云"
5. 点击"创建"，等待完成（约30秒）

### 3. 关联云服务空间
1. 右键点击 `uniCloud-aliyun` 文件夹
2. 选择 "关联云服务空间或项目"
3. 选择刚创建的空间
4. 点击"确定"

### 4. 初始化数据库
1. 右键点击 `uniCloud-aliyun/database` 文件夹
2. 选择 "上传 DB Schema 及扩展校验函数"
3. 等待上传完成

### 5. 完成！
现在运行项目：
```bash
npm run dev:h5
```

打开浏览器 → 进入"设置"页面 → 开启"云端同步"开关

## 🌐 方式二：使用 Web 控制台

### 1. 登录控制台
访问：https://unicloud.dcloud.net.cn/
（需要 DCloud 账号，免费注册）

### 2. 创建服务空间
1. 点击"新建服务空间"
2. 填写：
   - 空间名称：`lesson-tracking`
   - 服务商：阿里云
   - 套餐：免费版
3. 点击"创建"

### 3. 获取配置信息
1. 进入刚创建的服务空间
2. 点击"服务空间信息"
3. 复制 `SpaceID` 和 `SpaceAppId`

### 4. 配置项目
打开 `src/manifest.json`，找到 `uniCloud` 配置：

```json
{
  "uniCloud": {
    "provider": "aliyun",
    "aliyun": {
      "spaceId": "粘贴你的SpaceID",
      "spaceAppId": "粘贴你的SpaceAppId"
    }
  }
}
```

### 5. 创建数据库集合
在 Web 控制台：
1. 点击"云数据库"
2. 点击"新建集合"，创建：
   - 集合名：`lesson-course-packages`
   - 集合名：`lesson-records`

### 6. 完成！
运行项目，在"设置"中开启"云端同步"

## ❓ 故障排除

### 问题1：提示 "未关联服务空间"

**解决方法：**
1. 检查 `src/manifest.json` 中是否正确填写了 SpaceID
2. 重启开发服务器：
   ```bash
   # 按 Ctrl+C 停止
   npm run dev:h5
   ```

### 问题2：开启云同步后显示错误

**可能原因：**
- 网络连接问题
- 服务空间未创建成功
- 数据库集合未创建

**解决方法：**
1. 检查网络连接
2. 登录 Web 控制台验证服务空间是否存在
3. 查看浏览器控制台错误信息

### 问题3：数据不同步

**解决方法：**
1. 在"设置"页面点击"立即同步"
2. 检查是否开启了云同步开关
3. 查看同步状态提示

### 问题4：如何查看云端数据

1. 访问：https://unicloud.dcloud.net.cn/
2. 进入你的服务空间
3. 点击"云数据库"
4. 选择集合查看数据

## 📊 功能说明

### 开启云同步后：

✅ **自动同步**
- 打开应用时自动从云端拉取最新数据
- 添加/编辑数据时自动同步到云端
- 删除数据时自动从云端删除

✅ **离线可用**
- 无网络时数据保存在本地
- 恢复网络后自动同步

✅ **多设备共享**
- 手机、电脑数据实时同步
- 多人可共享同一数据

✅ **数据安全**
- 云端自动备份
- 本地缓存保证速度

### 关闭云同步后：

继续使用 localStorage 本地存储
- 数据仍然保存在本地
- 不会丢失任何数据
- 随时可重新开启
