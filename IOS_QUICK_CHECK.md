- [ ] **HBuilderX 已安装**
  - 版本：最新正式版
  - 已登录 DCloud 账号

  - 可以正常预览

## 🚀 打包步骤（5步）

### 第1步：打开打包界面

```
HBuilderX 菜单：
发行 → 原生App-云打包
```

### 第2步：选择平台

```
勾选：
☑ 仅打包 iOS
```

### 第3步：配置证书

#### 首次打包：

```
证书选择：
● 使用自有证书

上传文件：
- 证书私钥：选择 .p12 文件
- 证书密码：输入导出时设置的密码
- 证书profile：选择 .mobileprovision 文件
```

#### 已配置过：

```
证书选择：
○ 使用云端证书
选择之前上传的证书
```

### 第4步：确认配置

```
检查信息：
- Bundle ID：com.yourcompany.lessontracking
- 应用名称：课程学时登记
- 版本号：1.0.0
```

### 第5步：开始打包

```
点击：打包

等待时间：5-15分钟

查看进度：
HBuilderX 底部 → 云打包进度
```

## ❌ 常见错误及解决

### 错误1：证书不匹配

```
错误信息：
"Bundle identifier does not match"

解决方法：
1. 检查 manifest.json 中的 Bundle ID
2. 确保与证书的 App ID 完全一致
3. 重新打包
```

### 错误2：证书过期

```
错误信息：
"Certificate has expired"

解决方法：
1. 登录 Apple 开发者中心
2. 重新创建证书
3. 重新下载描述文件
4. 重新导出 .p12
5. 重新上传到 HBuilderX
```

### 错误3：描述文件不匹配

```
错误信息：
"Provisioning profile doesn't match"

解决方法：
1. 确保描述文件包含了正确的 App ID
2. 确保描述文件包含了当前的证书
3. 重新下载最新的描述文件
```

### 错误4：打包卡住不动

```
现象：
打包进度一直在 0%

解决方法：
1. 检查网络连接
2. 重启 HBuilderX
3. 清除云打包缓存
4. 重新打包
```

## 📱 安装后问题

### 问题1：应用打不开

```
现象：
点击图标后立即闪退

解决方法：
1. 检查证书是否过期
2. 查看 Xcode 控制台日志
3. 检查设备是否信任证书
   设置 → 通用 → VPN与设备管理 → 信任
```

### 问题2：提示"未受信任的企业级开发者"

```
解决方法：
设置 → 通用 → VPN与设备管理
→ 找到对应的开发者
→ 点击信任
```

### 问题3：功能异常

```
解决方法：
1. 使用 HBuilderX 真机调试
2. 查看控制台日志
3. 检查代码兼容性
4. 检查 manifest.json 配置
```
