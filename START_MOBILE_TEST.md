## 第1步：启动服务器

```bash
cd /Users/user/Projects/AI/lession-tracking

```

```
➜  Local:   http://localhost:5173/
```

## 第2步：手机访问

1. **确保手机和电脑在同一WiFi**

2. **iPhone 打开 Safari 浏览器**

3. **输入地址：**
   ```
   http://192.168.1.100:5173/
   ```
   （使用终端显示的 Network 地址）

4. **看到应用界面 = 成功！** ✅

## ❓ 如果访问不了

### 方法1：查看电脑IP地址

**Mac:**
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

会显示类似：
```
inet 192.168.1.100 netmask ...
```

用这个IP访问：`http://192.168.1.100:5173/`

**Windows:**
```bash
ipconfig
```

找到 "IPv4 地址"

### 方法2：检查防火墙

**Mac:**
```
系统偏好设置 → 安全性与隐私 → 防火墙
→ 确保允许 node 或 npm
```

### 方法3：确保同一WiFi

- 电脑和手机必须连接同一个WiFi
- 不能一个WiFi一个4G
