# uni-cloud 云数据库配置指南

## 📋 前置准备

1. **DCloud账号**：注册 https://dev.dcloud.net.cn/
2. **HBuilderX**：下载安装 https://www.dcloud.io/hbuilderx.html（推荐）
3. **或使用命令行**：已安装则跳过 HBuilderX

## 🚀 快速开始（3步完成）

### 方式一：使用 HBuilderX（推荐）

#### 第1步：创建云服务空间

1. 用 HBuilderX 打开项目
2. 右键点击 `uniCloud-aliyun` 目录
3. 选择 "创建云服务空间"
4. 填写空间名称（如：lesson-tracking）
5. 选择 "阿里云" 服务商
6. 点击 "创建"

#### 第2步：关联云服务空间

1. 右键点击 `uniCloud-aliyun` 目录
2. 选择 "关联云服务空间或项目"
3. 选择刚创建的空间
4. 点击 "确定"

#### 第3步：初始化数据库

1. 右键点击 `uniCloud-aliyun/database` 目录
2. 选择 "上传 DB Schema 及扩展校验函数"
3. 等待上传完成
4. 打开 uni-cloud web控制台验证：https://unicloud.dcloud.net.cn/

完成！现在可以开启云同步功能了。

---

### 方式二：使用 Web 控制台

#### 第1步：登录控制台

访问：https://unicloud.dcloud.net.cn/

#### 第2步：创建服务空间

1. 点击 "新建服务空间"
2. 填写空间名称
3. 选择 "阿里云"
4. 选择 "免费版"
5. 点击 "创建"

#### 第3步：获取 SpaceID

1. 进入刚创建的服务空间
2. 点击 "服务空间信息"
3. 复制 "SpaceID" 和 "SpaceAppId"

#### 第4步：配置项目

打开 `src/manifest.json`，填入 SpaceID：

```json
{
  "uniCloud": {
    "provider": "aliyun",
    "aliyun": {
      "spaceId": "你的SpaceID",
      "spaceAppId": "你的SpaceAppId"
    }
  }
}
```

#### 第5步：初始化数据库

在 Web 控制台：
1. 点击 "云数据库"
2. 点击 "新建集合"
3. 创建以下两个集合：
   - `lesson-course-packages`
   - `lesson-records`

## 📊 验证配置是否成功

### 方法1：在应用中测试

1. 运行应用：`npm run dev:h5`
2. 打开"设置"页面
3. 开启"云端同步"开关
4. 如果显示"已同步"则配置成功

### 方法2：查看控制台

1. 打开 https://unicloud.dcloud.net.cn/
2. 进入你的服务空间
3. 点击"云数据库"
4. 查看是否有两个集合

## 🎯 开始使用

### 1. 开启云同步

在应用"设置"页面，开启"云端同步"开关。

### 2. 自动同步

- **打开应用时**：自动从云端拉取最新数据
- **添加数据时**：自动同步到云端
- **编辑数据时**：自动同步到云端
- **删除数据时**：自动从云端删除

### 3. 手动同步

在"设置"页面，点击"立即同步"按钮。

## ❓ 常见问题

### Q1: 提示"未关联服务空间"

**解决方法：**
- 确认 `manifest.json` 中已填写正确的 SpaceID
- 重新启动开发服务器

### Q2: 同步失败

**可能原因：**
1. 网络问题 - 检查网络连接
2. 服务空间未创建 - 检查控制台
3. 权限问题 - 检查数据库权限配置

### Q3: 数据不一致

**解决方法：**
在"设置"页面，点击"立即同步"强制同步。

### Q4: 如何查看云端数据

1. 打开 https://unicloud.dcloud.net.cn/
2. 进入服务空间
3. 点击"云数据库"
4. 选择集合查看数据

## 🔒 数据安全

### 权限配置

当前配置：所有用户可读写（适合个人/家庭使用）

如需更严格的权限控制（多用户场景），修改 `database/*.schema.json`：

```json
{
  "permission": {
    "read": "auth.uid != null",  // 仅登录用户可读
    "create": "auth.uid != null", // 仅登录用户可写
    "update": "auth.uid != null",
    "delete": "auth.uid != null"
  }
}
```

### 数据备份

建议定期在"设置"页面导出数据备份。

## 📝 注意事项

1. ✅ **本地优先**：即使云同步失败，数据仍保存在本地
2. ✅ **断网可用**：离线时所有功能正常，联网后自动同步
3. ✅ **数据安全**：云端使用阿里云，安全可靠
4. ⚠️ **首次同步**：首次开启云同步会上传所有本地数据到云端
5. ⚠️ **删除空间**：删除云服务空间会永久删除云端数据
