# 费用功能快速上手指南

## 🎯 一分钟快速导入历史数据

### 步骤1：启动应用
```bash
npm run dev:h5
```

### 步骤2：导入历史费用
1. 打开浏览器访问应用（通常是 http://localhost:5173）
2. 点击底部导航栏的"设置"
3. 找到"💰 费用管理"区域
4. 点击"导入历史费用"
5. 确认导入
6. ✅ 完成！系统会自动添加你的4笔历史支付记录

### 步骤3：查看费用统计
- 返回首页，你会看到"总支出: ¥111,600.00"
- 点击"费用"按钮查看详细记录

## 🔥 常用功能

### 1. 查看费用记录
- **入口1**：首页 → 费用按钮
- **入口2**：设置 → 费用记录
- **显示内容**：
  - 总支出金额
  - 支付次数
  - 每笔详细记录（日期、课程包、金额）

### 2. 添加新的支付
1. 进入"费用记录"页面
2. 点击"+ 添加支付"
3. 选择日期
4. 添加课程包（可添加多个）
5. 填写备注（可选）
6. 提交

### 3. 编辑支付记录
- 在费用列表中点击"编辑"按钮
- 修改日期、课程包或备注
- 保存

### 4. 删除支付记录
- 在费用列表中点击"删除"按钮
- 确认删除

## ⚠️ 注意事项

1. **数据安全**
   - 所有数据存储在本地浏览器
   - 建议定期导出备份
   - 清除浏览器缓存会丢失数据

2. **防重复导入**
   - 系统自动检测重复记录
   - 相同日期的记录不会重复导入

3. **数据导出**
   - 设置 → 数据管理 → 导出数据
   - 会下载包含所有数据的JSON文件

## ❓ 常见问题

**Q: 导入后没有看到数据？**
A: 刷新页面或重新进入费用记录页面

**Q: 可以修改已导入的历史数据吗？**
A: 可以，点击"编辑"按钮即可修改

**Q: 费用记录和课程包是关联的吗？**
A: 目前是独立管理的，费用记录用于财务统计，课程包用于学时管理

**Q: 如何备份数据？**
A: 设置 → 数据管理 → 导出数据，会下载JSON文件
