# 快速启动指南

## 安装步骤

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

#### H5 开发模式
```bash
npm run dev:h5
```

启动后访问：http://localhost:5173

#### APP 开发模式（需要 HBuilderX）
```bash
npm run dev:app
```

## 功能演示

### 场景1：添加第一个课程包

1. 打开应用，进入"课程包"页面
2. 点击"添加课程包"
3. 填写信息：
   - 课程类型：认知课程
   - 课程名称：认知训练
   - 总学时：48
   - 购买日期：2023-11-28
4. 点击"添加"

### 场景2：记录每日学时消耗

1. 进入"首页"或"学时登记"页面
2. 选择课程包：认知训练
3. 选择记录类型：消耗学时
4. 输入消耗学时：1
5. 选择日期：2024-01-25
6. 点击"提交"
7. 查看剩余学时：47/48

### 场景3：记录请假

1. 进入"学时登记"页面
2. 选择课程包：认知训练
3. 选择记录类型：请假
4. 选择日期：2024-01-26
5. 添加备注：临时有事
6. 点击"提交"

### 场景4：查看历史记录

1. 进入"记录"页面
2. 可以看到按日期分组的所有记录
3. 点击筛选器可以按课程包筛选
4. 可以删除错误的记录

## 常见问题

### Q1: 为什么看不到课程包？
A: 首次使用需要先添加课程包，点击"添加课程包"开始。

### Q2: 如何修改错误的记录？
A: 在历史记录页面删除错误记录，然后重新添加正确的记录。

### Q3: 删除记录会发生什么？
A: 删除记录会自动恢复对应的学时到课程包中。

### Q4: 数据会丢失吗？
A: 数据保存在本地存储中，除非清除浏览器数据或应用数据，否则不会丢失。

### Q5: 可以导出数据吗？
A: 当前版本暂不支持导出功能，后续版本会添加。

## 开发建议

### 调试技巧

1. 使用浏览器开发者工具查看本地存储：
   - Application -> Local Storage
   - 查看 `lesson_tracking_` 开头的数据

2. 清除测试数据：
   ```javascript
   // 在浏览器控制台执行
   localStorage.clear()
   ```

### 自定义配置

#### 修改课程类型
编辑 `src/constants/course.ts` 文件，添加新的课程类型。

#### 修改主题色
编辑 `src/App.vue` 文件，修改全局样式中的颜色变量。

## iOS APP 打包流程

### 准备工作
1. 安装 HBuilderX
2. 准备 Apple 开发者账号
3. 准备 iOS 证书和描述文件

### 打包步骤
1. 用 HBuilderX 打开项目
2. 点击"发行" -> "原生App-云打包"
3. 选择 iOS 平台
4. 填写应用信息：
   - AppID
   - 证书
   - 描述文件
5. 点击"打包"
6. 等待打包完成
7. 下载 ipa 文件

### 安装测试
1. 使用 Xcode 或第三方工具安装 ipa
2. 在真机上测试功能

## 性能优化建议

1. **数据量大时的优化**
   - 当记录超过 1000 条时，考虑添加分页功能
   - 可以添加数据归档功能，归档旧数据

2. **加载速度优化**
   - 使用虚拟列表展示大量记录
   - 懒加载历史记录

3. **存储优化**
   - 定期清理无用数据
   - 压缩存储数据
